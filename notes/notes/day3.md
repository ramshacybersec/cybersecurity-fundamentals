# Day 3 - OSI Model

## What is OSI Model?
The OSI Model is a framework used to understand how data travels in a network.

## 7 Layers
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## What I Learned
Today I learned how network communication is divided into layers and how each layer has a specific function.
